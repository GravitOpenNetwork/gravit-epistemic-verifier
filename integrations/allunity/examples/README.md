# Agentic Payment Examples

This folder contains three reference scenarios:

1. **A2S Micropayment** — Agent pays for API call (0.5 EURAU)
2. **A2M Delegated** — Agent buys SaaS license with human mandate
3. **A2A Negotiation** — Two agents trade compute resources

## To run

```bash
python a2s_micropayment.py
python a2m_delegated.py
python a2a_negotiation.py
