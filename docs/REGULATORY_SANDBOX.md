# Regulatory Sandbox Proposal (BaFin / FINMA)

## Objective

Joint pilot with AllUnity to demonstrate epistemic verification for agentic payments within a regulatory sandbox.

## Proposed Scope

- **Duration:** 6 months
- **Volume:** Limited (e.g., 1000 transactions / day)
- **Participants:** AllUnity (EURAU, x402) + Gravit (epistemic verifier)
- **Regulator Oversight:** BaFin (Germany) or FINMA (Switzerland)

## What We Will Show

1. Agentic payment (A2S model) with Gravit pre-settlement verification
2. Full audit trail generation
3. Adversarial attack detection (sandwich, replay, intent substitution)
4. Regulatory dashboard with real-time compliance metrics

## Deliverables for Regulator

- Weekly compliance reports (machine-readable + human-readable)
- API access to verification logs (read-only)
- Final report with recommendations for MiCAR implementation

## Next Step

Submit formal sandbox application jointly with AllUnity after PoC success.
